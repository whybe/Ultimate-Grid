=============================================
Ultimate Grid for MFC Development Kit
Simple Checklist wrapper
=============================================


This program demonstrates how easy it is to create custom
grid controls using the Ultimate Grid for MFC Development Kit.

The CChecklistGrid class is derived from CUGCtrl and provides
an easy way to maintain a checklist.  Users can specify several
different styles, like check marks or crosses, borders, and more.

Please read the class definition for more information about the
functions and how to use the class.


Unzip the file to your \Examples directory.  This program will
search for the necessary include and lib files in ..\..\include 
for the inlcude files and ..\..\vc50 for the lib file.
