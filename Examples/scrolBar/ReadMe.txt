      UG97 Scroll mode 
/////////////////////////////////////

This example demonstrates the Scroll bars Modes
to change the vertical scrolling mode 	

	SetVScrollMode(UG_SCROLLNORMAL);

Other modes 
	UG_SCROLLTRACKING
	UG_SCROLLJOYSTICK

to hide the vertical scroll bar :
	SetVS_Width(0);

to change the horizontal scrolling mode 	
	SetHScrollMode(UG_SCROLLNORMAL);
	
	UG_SCROLLTRACKING
	UG_SCROLLJOYSTICK

to hide the horizontal scroll bar :
	SetHS_Height(0);
