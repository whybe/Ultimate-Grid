      Ballistic Modes
/////////////////////////////////////

This example illustrates the ballistic scrolling
modes possible in Ultimate Grid 97.


This example demonstrates 	SetBallisticMode() function 
when running this example, while you are in the grid client area hold down the left
mouse button and drag it in any direction way pass the client area of the grid.
The furthure from the grid the mouse pointer is the faster the grid 		

To change the mode use 	SetBallisticMode(0) // the modes are 0 ,1, 2 , or 3
To Change the Delay os scrolling use SetBallisticDelay(100); // 100 milisecond
	
	

Examine code in MyCug::OnMenuCommand() for details.
